# 简介
* Nestjs
* TypeScript
* MySQL
* JWT登录鉴权
* typeorm自动建表

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
